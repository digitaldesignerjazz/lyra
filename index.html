GET STARTED IN SECONDS

Run your first Lyra agent

The CLI is fully functional today. Launch single agents or coordinated swarms with emotional messaging.

TERMINAL 
git clone https://github.com/digitaldesignerjazz/lyra.git
cd lyra

python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

# Run a single agent
lyra --name "Lyra-Prime" --mode single --steps 8

# Launch an emotional swarm
lyra --name "Nova" --mode swarm --agents 5 --steps 10 --verbose

The core emotional agent runtime, memory system, and inter-agent messaging are already working and demoable.

Implemented

  • Pydantic-based LyraAgent with emotional state
  • Tiered memory (short-term, long-term, reflections)
  • Inter-agent messaging with emotional modulation
  • Self-improvement loops
  • Full-featured CLI (single + swarm modes)
  • Modern Python packaging (pyproject.toml + hatch)
